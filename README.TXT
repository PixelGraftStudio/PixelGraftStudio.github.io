Nombre del proyecto

--Compra de Entradas - Maluma--

Descripción

Este proyecto consiste en una página web estática que permite a los usuarios consultar fechas de conciertos y comprar entradas para un artista musical (en este caso, Maluma).
 El sitio está compuesto por dos páginas:

index.html: página principal donde se muestra la imagen del grupo, la descripción, fechas de conciertos y un formulario de compra.

confirmacion.html: página de confirmación donde se visualizan los datos ingresados por el usuario después del envío del formulario.

Funcionalidades:

Página principal con diseño responsive.

Imagen del grupo centrada y redondeada.

Tabla con fechas de conciertos (alternancia de colores).

Formulario con validación de campos.

Almacenamiento de datos en sessionStorage.

Redirección a página de confirmación.

Visualización clara y estructurada de los datos enviados.

Compatible con dispositivos móviles.

Créditos y Fuentes de Información

Texto descriptivo del grupo: tomado y adaptado desde la web de El Corte Inglés.

Información general y fechas ficticias: inspiradas en ejemplos de entradas vistas en portales web.

Recursos visuales (imágenes): obtenidas desde Google Imágenes.

Ayuda para estructura HTML/CSS y validación con JavaScript: investigado en YouTube,libro de lenguaage de marcas y sistemas de gestiones,vt,www.w3schools.com
